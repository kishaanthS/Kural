# TN News Hub

A mobile-optimized news application focusing on Tamil Nadu Politics, India, and World News.

## Key Features
- **TN Focus**: Dedicated sections for Tamil Nadu Politics and regional news.
- **AI Powered**: Uses Gemini AI with real-time Google Search to fetch and summarize the latest stories.
- **Mobile First**: Designed as a Progressive Web App (PWA) with a native mobile look and feel.
- **Smart Summaries**: Concise summaries for quick consumption.

## Installation (Android/iOS)
Since this is a Progressive Web App, you can install it on your Android device without an APK:
1. Open the App URL in Chrome on your Android device.
2. Tap the three dots (menu) in the top right corner.
3. Select **"Add to Home screen"** or **"Install App"**.
4. The TN News Hub icon will now appear on your home screen and function like a native app.

## APK Generation
To generate a native APK for the Play Store, you can use services like **Bubblewrap** or **PWA2APK** using the manifest file provided in this project.
